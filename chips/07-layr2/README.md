# 07-layr-2

> [!CAUTION]
> This is not our final submission. We intend to implement Layr level 1 for the actual tapeout.

## Authors

Valentin Buttner, Florian Herrnberger, Emilia Jaser, Felix Kreil, Julian Rapp

## Description

The current implementation covers approximately level 0.5 of the [Layr Challenge](https://layr-challenge.de/). Most notably, AES support is not yet implemented.

## Pin List

Pins match the QFN package specified [here](https://github.com/OCDCpro/LAYR/tree/main/challenge) by the Layr Challenge, with one deviation: UART is not implemented, so no UART pins are included in the design.
